near-runtime-ts

Typescript library for writing near smart contracts

[

Generate documentation
----------------------

](#generate-documentation)

API docs are produced using typedoc. To run, first install dependencies:

```
npm install
```

After installing dependencies, run

```
npm run doc
```

[

Run tests
---------

](#run-tests)

```
npm test
```
